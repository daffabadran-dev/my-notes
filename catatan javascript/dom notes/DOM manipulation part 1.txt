classList:
elemet.classList.add()		
----------------.remove()
		.toggle()
		.item()
		.contains()
		.replace()


- add
untuk menambahkan class baru

- remove
menghapus class 

- toggle
jika ada class hapus, jika tidak ada class tambahkan, sama seperti saklar

- item
untuk mengetahui class tertentu, misal elemen punya 3 class, aku ingin tahu class ke 3

- contains
untuk mengecek dalam sebuah elemen apa memiliki class tertentu

- replace
mengganti class
