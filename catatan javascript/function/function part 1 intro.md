-> user define function

\- fungsi yang kita buat sendiri

\- menggunakan keyword function

\- nama function (opsional)

\- parameter / argumen

 	- disimpan di dalam ()

 	- boleh ada atau tidak, jika ada boleh lebih dari satu (isi argumen yang di isikan ke parameter)

 	- dipisahkan oleh , (koma)

\- dapat mengembalikan nilai (return value) atau tidak









-> membuat  user-defined function:



 - dengan deklarasi

      //function deklrasi

      function jumlahDuabilangan(a, b) {

        return a + b;

      }

      console.log(jumlahDuabilangan(3, 10));

 

 - dengan ekspresi

      //function ekspresi

      let kaliDua = function (angka) {

        return angka \* 2;

      };

      console.log(kaliDua(5));
