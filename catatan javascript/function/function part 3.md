# function part 3 parameter dan argument



parameter adalah variabel yang ditulis di dalam kurung contoh function kerupuk (parameter) {}





argument adalah nilai yang dikirimkan pada parameter



contoh:

 	function tambah (a, b) { //ini adalah paramter

 	return a + b;

 	}

 	let coba = tambah (5, 10); //ini adalah argument

 	console.log(coba)
