# array

"tipe data yang digunakan untuk mendeskripsikan kumpulan elemen, yang tiap tiap elemennya memiliki sesuatu yang disebut dengan index"



"array itu adalah sebuah variabel tapi lebihs sakti, karena bisa menampung lebih dari 1 nilai, bisa menyimpan angka, string, Boolean"



#### kenapa array?

* mempermudah pengelolaan nilai / value / data
* penelusuran dan pencarian
* manajemen memori



array:

* variabel jamak
* kumpulan pasangan key and value pair
* key adalah index pada array dengan tipe data integer yang dimulai dari 0
* array pada JS BERTIPE OBEJECT
* array pada js memiliki fungsi/method 
* elemen pada array boleh memiliki tipe data yang berbeda
