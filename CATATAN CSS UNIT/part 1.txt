CSS VALUE

nilai yang bisa kita beri:
- integer
- number
- percentage
- dimension

integer: nilai positif dan negatif, no koma komaan

number: ya angka, bisa desimal
persen: %

dimension/dimensi:
number yang memiliki satuan di belakangnya

- length
- angle
- time
- resolution


ANGLE / SUDUT
- deg (degree), derajat
- rad (radiant)
- grad (gradiant)
- turn (untuk menentukan berapa putaran)

TIME
- s (second)
- ms (milisecond)


RESOLUTION
- dpi (dot per inch)
- dpcm (dot per cm)
- dppx (dot per pixel)

@media print and (min-resoulution: 300dpi) {}


LENGTH:
- absolute length / fix
- relative
